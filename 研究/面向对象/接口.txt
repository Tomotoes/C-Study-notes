接口：

接口定义了所有类继承接口时应遵循的语法合同，

接口定义了语法合同“是什么 ”部分

派生类定义了语法合同“怎么做 ”部分

接口定义了属性，方法，事件，这些都是接口的成员。

接口只包含成员的声明。

成员的定义是派生类的责任。

接口提供了派生类应遵循的标准结构。

接口 是引用类型 

接口 默认是 public 不能加任何修饰符！

接口的强大之处 ：可以继承多个接口

  interface A
    {
        void Q();    // 注意不能加任何修饰符！只能是单纯的函数声明。
    }
    class B : A
    {
      public void Q()//注意实现类中 反而应该加 public修饰符，因为接口默认 public
      {
            Console.WriteLine("Heall!");
      }
    }

使用： 
	B b = new B();
        (b as A).Q();
